# SearchGithubUsers

This app has be build using:

- 100% Kotlin
- Added extra layer of security to get Base url using C++
- MVVM
- Hilt for DI
- Navigation Component
- Flow for reactive programming
- ViewBinding for binding
- Coil for Images
- Retrofit for network
- Moshi
- Paging 3 for pagination
- Espresso
- Timber
- JUnit4
